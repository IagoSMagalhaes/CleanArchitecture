## Grooming


O termo se refere à preparação de backlog  (que são os requisitos ou lista de pendências dentro do Scrum) e é mais adequadamente conhecido como refinamento. É o ato de detalhar, entender mais profundamente, adicionar características, estimar, priorizar e manter o backlog do produto vivo.

#### Padrão de logs

        runCatching {
            LOG.info("START {} user: {} body {}", methodName/service?, cpf, body)

            service.execute();

        }.onFailure {
            LOG.error("ERROR {} user: {} statusCode: {} cause: {} message: {}", methodName/service?, cpf, statusCodeResponse, causeResponse, messageResponse)
            
            throw it

        }.onSucesss {
            LOG.info("END {} user: {}", methodName/service?, cpf)
        }
